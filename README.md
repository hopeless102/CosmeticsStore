# BeautyShop - Магазин косметики

## Описание
Интернет-магазин косметики для продажи товаров по категориям:
- Уход за кожей
- Декоративная косметика
- Парфюмерия
- Мужская косметика

## Технологии
- C# .NET 6.0 Windows Forms
- JSON для хранения данных
- Git для контроля версий

## Функционал
- Просмотр каталога с поиском и фильтрацией
- Корзина с добавлением/удалением товаров
- Оформление заказа с валидацией данных
- Административная панель (пароль: admin123)

## Запуск
Откройте файл `CosmeticsStore.sln` в Visual Studio и нажмите F5
